# Amazon-Sales-BI-Dashboard-
Amazon Sales Business Intelligence Command Center

 Project Overview -

This project presents an end-to-end Business Intelligence solution built using Amazon sales data. The objective is to analyze sales performance, customer behavior, product trends, and generate actionable insights to support business decision-making.

 Objectives -

- Analyze sales and revenue trends
- Understand customer behavior and segmentation
- Evaluate product and category performance
- Identify cross-selling and upselling opportunities
- Perform demand forecasting and seasonal analysis
- Build an executive command center dashboard

 Tools & Technologies Used

- Python (Pandas) – Data cleaning & preprocessing
- SQL – Data storage and querying
- Power BI – Data modeling and dashboard development
- DAX – KPI calculations and dynamic measures

 Project Workflow

1. Data Cleaning & Preprocessing using Python
2. Data storage and transformation using SQL
3. Data modeling using Star Schema in Power BI
4. Dashboard development with interactive visuals
5. Insight generation and business analysis

 Dashboards Developed

- Sales & Performance Overview
- Customer & Pricing Intelligence
- Customer Retention & Product Performance
- Product & Operations Analytics
- Operations & Predictive Intelligence
- Strategic Business Intelligence Command Center

 Key Features

- 25+ interactive visualizations
- KPI monitoring and performance tracking
- Forecasting and scenario analysis
- Cross-sell and upsell insights
- Geographic customer analysis
- Seasonal demand planning

Key Insights

- Metro cities dominate customer base and revenue
- Premium products drive higher average order value
- Seasonal demand significantly impacts sales performance
- Cross-selling opportunities exist between smartphones and accessories
- Customer retention and satisfaction levels are high

 Business Impact

- Enables data-driven decision-making
- Helps identify growth opportunities
- Supports marketing and inventory strategies
- Improves operational efficiency
